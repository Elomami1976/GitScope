About GitScope

What is GitScope?

GitScope is a browser-based GitHub repository visualizer that helps you explore and understand any public repository's structure, contributors, and history — all without leaving your browser.

Features

  • Interactive Treemap: Visualize file sizes and commit activity with drill-down navigation
  • Contributor Insights: See who contributes most and identify "bus factor" risks
  • File Intelligence: Find abandoned files, most changed files, and largest files
  • Language Breakdown: View repository language composition
  • Search & Filter: Filter by author, file type, or date range
  • Export: Save treemap as PNG or share via URL

Technology

GitScope is built entirely with vanilla JavaScript — no frameworks, no build steps. It uses D3.js for treemap visualizations and Chart.js for charts.

API Rate Limits

GitScope uses the unauthenticated GitHub API, which allows 60 requests per hour. Each repository analysis uses approximately 6-8 requests. For best results, use on small to medium-sized repositories.
